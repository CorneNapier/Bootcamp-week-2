Static(Default)

Default position property for elements

Relative

Allows to move element via top, left, right, bottom properties, relative to the elements starting position.

Absolute

Positions the element absolutely inside its closest NON-Static postioned element. Otherwise it will be positioned absolutely in the viewport.

Fixed

Fixes an element to a postion on the viewport, which will stay on viewport whether you scroll down or shrink viewport.

Sticky

Similar to fixed, however it will stay in its natural position, but will stick the element to a specified point when scrolled past.
