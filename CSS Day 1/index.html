Box-Model

Each HTML element in CSS is essentially a box. A box is made up of 4 sections; Margin, Border, Padding, Content.

  • Margin - The space between the element and the page border, or the surrounding elements.
  • Border - The border that surrounds the padding and content.
  • Padding - The space between the content and the border. Even if the border is set to 0, padding still sits between it and the content.
  • Content - The content is where the text or image will appear. The size of the content is determined by the height and width properties.
